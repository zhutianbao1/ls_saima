会员须知:

热爱读书,有志于提高本人的读书水平,同时愿意为他人推荐好书以 及分享自己心得体会的在册员工,都可申请成为读书会会员.

(一)会员的权力:
  1. 1.参加读书会的会议.活动,参加读书会发展的讨论, 对读书会的工作提出意见和倡议.
  2. 2.分享读书的经验,优先获取读书会最新资讯,享受读书会的有关服务.
  3. 3.优先享有读书会对学习成果的推荐.
  4. 4.优先参加公司举办的相关学习培训.
  5. 5.在读书会内有表决权,选举权和被选举权,有批评权,建议权和监督权.
(二)会员的义务:
  1. 1.读书会会员要多读书,读好书,自觉保持和养成良好的读书习惯. 每天坚持阅读10分钟以上,精读一本当月推荐书,并推荐给两位以上非会员.
  2. 2.主动和他人分享阅读经验.自入会之日起, 每季度至少参加一次读书交流活动或提交一篇读书体会.
  3. 3. 读书会会员要自觉遵守读书会章程,能认真执行读书会的每一项决议和倡导,积极参与读书会的各项活动.
  4. 4.维护读书会的荣誉,宣传读书会的宗旨,发展读书会的新会员.
  5. 5.自觉遵守国家的法律法规,自觉遵守社会公德, 认真履行好个人承担的工作学习任务和社会责任.
(三)积分规则:
  1. 1.读书会会员参与所在单位组织的月度读书会活动并分享读书心得 或看完月度推荐书籍并提交心得,得5分.
  2. 2.阅读每月必读推荐书籍之外的书籍并提交心得的或参加读书会并 分享读书心得的,得3分.
  3. 3.参与月度读书会活动主持、策划、讲课,每人额外得5分, 每期活动主持策划最多不超过2人.
  4. 4.员工推荐书籍采纳为公司推荐书籍的,得2分.
